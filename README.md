# Locker Status Application

This web application displays the status of lockers in a grid format. The status is represented by different colors, and the grid updates every 60 seconds by fetching data from an API.

## Features

- **Grid Display**: Displays lockers in a 13x8 grid with color-coded status.
- **Real-time Updates**: Data is fetched from an API every 60 seconds to keep the grid updated.
- **Tooltips**: Hover over grid items to see detailed information about each locker.
